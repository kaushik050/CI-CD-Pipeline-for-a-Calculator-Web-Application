# CI/CD Calculator

A simple web-based calculator built with Flask, styled using CSS with glowing effects.

## Features
- Addition, Subtraction, Multiplication, Division
- Beautiful dark glowing theme
- Unit tested with Pytest
- Ready for CI/CD pipeline integration

## Run Locally

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
